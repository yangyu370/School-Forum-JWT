# 快速开始指南

本指南将帮助你在10分钟内快速启动校园论坛系统。

---

## 📋 前置准备

### 必需软件
- [x] JDK 17 或更高版本
- [x] Node.js 16 或更高版本
- [x] MySQL 8.0 或更高版本
- [x] Redis 7.0 或更高版本
- [x] Maven 3.6 或更高版本

### 可选软件
- [ ] IntelliJ IDEA (推荐用于后端开发)
- [ ] VS Code (推荐用于前端开发)
- [ ] MinIO Server (用于图片存储)
- [ ] RabbitMQ (用于邮件队列)

## 🔧 环境安装

### 1. 安装 JDK 17
```bash
# macOS
brew install openjdk@17

# Windows
# 从 Oracle 官网下载安装包：https://www.oracle.com/java/technologies/downloads/

# 验证安装
java -version
```

### 2. 安装 Node.js
```bash
# macOS
brew install node

# Windows
# 从 Node.js 官网下载安装包：https://nodejs.org/

# 验证安装
node -v
npm -v
```

### 3. 安装 MySQL
```bash
# macOS
brew install mysql
brew services start mysql

# Windows
# 从 MySQL 官网下载安装包：https://dev.mysql.com/downloads/

# 设置root密码
mysql_secure_installation
```

### 4. 安装 Redis
```bash
# macOS
brew install redis
brew services start redis

# Windows
# 从 Redis 官网下载安装包：https://redis.io/download

# 验证安装
redis-cli ping
# 应返回 PONG
```

### 5. 安装 Maven
```bash
# macOS
brew install maven

# Windows
# 从 Maven 官网下载安装包：https://maven.apache.org/download.cgi

# 验证安装
mvn -v
```

## 🗄️ 数据库配置

### 1. 创建数据库
```sql
-- 登录MySQL
mysql -u root -p

-- 创建数据库
CREATE DATABASE campus_forum CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 创建用户（可选）
CREATE USER 'forum_user'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON campus_forum.* TO 'forum_user'@'localhost';
FLUSH PRIVILEGES;

-- 退出
EXIT;
```

### 2. 导入数据表

找到项目中的 SQL 初始化脚本并执行：
```bash
mysql -u root -p campus_forum < database.sql
```

或者在启动项目时，MyBatis Plus 会自动创建表结构（如果配置了自动建表）。

## 🚀 后端启动

### 方式一：使用 IDEA（推荐）

1. **打开项目**
   - 启动 IntelliJ IDEA
   - 选择 `Open` → 找到 `my-project-backend` 文件夹
   - 等待 Maven 下载依赖

2. **配置数据库**
   
   编辑 `src/main/resources/application.yml`：
   ```yaml
   spring:
     datasource:
       url: jdbc:mysql://localhost:3306/campus_forum?useSSL=false&serverTimezone=Asia/Shanghai
       username: root
       password: your_password
     
     data:
       redis:
         host: localhost
         port: 6379
         password:  # 如果设置了密码则填写
   ```

3. **运行项目**
   - 找到 `MyProjectBackendApplication.java`
   - 右键选择 `Run 'MyProjectBackendApplication'`
   - 等待启动完成

### 方式二：使用命令行

```bash
cd my-project-backend

# 安装依赖
mvn clean install -DskipTests

# 启动项目
mvn spring-boot:run
```

### 验证后端启动成功

访问：http://localhost:8080/swagger-ui/index.html

如果看到 Swagger 文档页面，说明后端启动成功！

## 🎨 前端启动

### 1. 安装依赖

```bash
cd my-project-frontend

# 使用 npm 安装
npm install

# 或使用 yarn
yarn install

# 或使用 pnpm（更快）
pnpm install
```

### 2. 配置后端地址

编辑 `vite.config.js`（通常已配置好，无需修改）：
```javascript
export default defineConfig({
  server: {
    port: 5173,
    proxy: {
      '/api': {
        target: 'http://localhost:8080',
        changeOrigin: true
      }
    }
  }
})
```

### 3. 启动开发服务器

```bash
npm run dev
```

### 4. 访问前端

浏览器自动打开 http://localhost:5173

如果没有自动打开，手动访问该地址。

## 📧 邮件服务配置（可选）

如果需要注册功能，需要配置邮件服务。

### 使用 QQ 邮箱

1. **开启 SMTP 服务**
   - 登录 QQ 邮箱
   - 设置 → 账户 → POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务
   - 开启 "POP3/SMTP服务" 或 "IMAP/SMTP服务"
   - 生成授权码（保存好）

2. **修改配置文件**
   
   编辑 `application.yml`：
   ```yaml
   spring:
     mail:
       host: smtp.qq.com
       port: 465
       username: your_email@qq.com
       password: 你的授权码（不是邮箱密码）
       properties:
         mail:
           smtp:
             ssl:
               enable: true
   ```

### 使用 163 邮箱

```yaml
spring:
  mail:
    host: smtp.163.com
    port: 465
    username: your_email@163.com
    password: 你的授权码
```

## 📦 MinIO 配置（可选）

用于图片存储，如果暂时不需要，可以跳过。

### 1. 安装 MinIO

```bash
# macOS
brew install minio/stable/minio

# 启动 MinIO
minio server /data

# 或使用 Docker
docker run -p 9000:9000 -p 9001:9001 \
  -e "MINIO_ROOT_USER=minioadmin" \
  -e "MINIO_ROOT_PASSWORD=minioadmin" \
  minio/minio server /data --console-address ":9001"
```

### 2. 访问 MinIO 控制台

- 访问：http://localhost:9001
- 用户名：`minioadmin`
- 密码：`minioadmin`

### 3. 创建存储桶

在 MinIO 控制台中创建一个名为 `campus-forum` 的存储桶。

### 4. 配置项目

编辑 `application.yml`：
```yaml
minio:
  endpoint: http://localhost:9000
  accessKey: minioadmin
  secretKey: minioadmin
  bucket: campus-forum
```

## 🐰 RabbitMQ 配置（可选）

用于异步邮件发送。

### 1. 安装 RabbitMQ

```bash
# macOS
brew install rabbitmq
brew services start rabbitmq

# 或使用 Docker
docker run -d --name rabbitmq \
  -p 5672:5672 -p 15672:15672 \
  rabbitmq:3-management
```

### 2. 访问管理界面

- 访问：http://localhost:15672
- 用户名：`guest`
- 密码：`guest`

### 3. 配置项目

编辑 `application.yml`：
```yaml
spring:
  rabbitmq:
    host: localhost
    port: 5672
    username: guest
    password: guest
```

## 🔍 常见问题排查

### 后端无法启动

**问题1：端口被占用**
```bash
# 查看8080端口占用情况
lsof -i:8080  # macOS/Linux
netstat -ano | findstr 8080  # Windows

# 修改端口（在application.yml中）
server:
  port: 8081
```

**问题2：数据库连接失败**
- 检查MySQL是否启动
- 检查用户名密码是否正确
- 检查数据库名称是否正确
- 检查MySQL端口是否为3306

**问题3：Redis连接失败**
```bash
# 检查Redis是否启动
redis-cli ping

# 如果没启动
brew services start redis  # macOS
redis-server  # Windows
```

### 前端无法启动

**问题1：依赖安装失败**
```bash
# 清除缓存重新安装
rm -rf node_modules
npm cache clean --force
npm install
```

**问题2：端口被占用**
```bash
# 修改端口（在vite.config.js中）
export default defineConfig({
  server: {
    port: 5174  // 改为其他端口
  }
})
```

### 邮件发送失败

- 检查邮箱配置是否正确
- 确认使用的是授权码而不是邮箱密码
- 检查防火墙是否拦截了SMTP端口
